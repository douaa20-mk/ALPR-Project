Automatic License Plate Recognition (ALPR)

Internship Project at BMT (Bejaia Mediterranean Terminal)

ğŸ“˜ Project Description

We used the YOLOv8n pre-trained model and created a dataset of 7,829 images labeled with Roboflow. To build our dataset, we combined different open-source datasets from Kaggle, Roboflow, and other websites.

Why YOLOv8?

YOLOv8 is the latest version of the "You Only Look Once" object detection models. It is fast, accurate, and lightweight, making it suitable for real-time applications like license plate detection. YOLO works by dividing an image into a grid and predicting bounding boxes and class probabilities for each grid cell in a single forward pass.

Why Roboflow?

Roboflow simplifies labeling, augmenting datasets, and exporting them in YOLO-compatible formats. This saved us time and allowed us to focus on model training and evaluation.

Training Approach

  • Model: YOLOv8n (nano version for speed)
  • Dataset: 7,829 labeled images
  • Tools: Roboflow, Kaggle datasets, other open-source resources
  • Goal: High accuracy while maintaining real-time detection performance
